# LABS 改善計画 — ゲーム性と軌道力学学習の両立

作成: 2026-06-10

## 0. 現状診断

**既に良い点(残すべき資産):**

- HCW 方程式そのものは正しく実装済み(V/R/H のスクリーン写像も整合)
- `r × F` の 3D トルク + レイキャストによる面ヒット判定は「どこを撃つと回転が
  止まる/悪化するか」を体感させる良い設計
- 太陽・食モデルは β 角・食半角の式が本物で、エネルギー管理のリズムを作れている

**学習効果を打ち消していた 3 つの設計(Sprint 1 で解消):**

1. **人工減衰** — 速度・角速度に毎フレーム減衰。真空軌道は無減衰であることこそ
   HCW の核心なのに、「放っておけば落ち着く」という誤った直感を植え付ける。
2. **画面端の壁バウンド** — 相対運動の最重要レッスンは「制御しないと漂流して
   戻ってこない」なのに、壁が勝手に戻してくれる。
3. **「下に押せばデオービット」という勝利条件** — 物理的に誤った教訓。HCW では
   ラジアル(−R)方向のインパルスは閉じた 2:1 楕円を描いて 1 周期後に戻ってくる
   (永年ドリフトなし)。軌道を下げるのはリトログレード(−V)方向の Δv であり、
   その帰結は「降下しながら前方へドリフト」というサイクロイド。

→ 最高レバレッジの改善は「この 3 つを直し、その帰結が見える R–V 面の可視化を
足す」こと。ゲームの戦略性(Δv をどの方向に・いつ入れるか)と学習効果が同時に
跳ね上がる。

### Sprint 1 設計判断(実装メモ)

- **座標系**: ヒル/LVLH 座標 `rel = {x: radial(+zenith), y: along-track(+prograde),
  z: cross-track}` [m]。基準軌道原点 = チェイサー初期位置。
  描画用スクリーン座標は `rel` から毎フレーム導出(レンダリングコードは不変更)。
- **時間**: 実平均運動 n = √(μ/a³) ≈ 1.085×10⁻³ rad/s (600 km)。
  `TIME_WARP = 64` で 1 周回 ≈ 90 壁時計秒(従来のペーシングを維持)。
  並進・軌道・太陽 = 物理時間、姿勢回転・熱・エネルギー = 従来のゲーム時間
  (実機のタンブル除去は時間スケールが桁で違うため、回転は様式化を継続。
  幾何・トルク符号は正確)。
- **伝播**: CW 閉形式解(状態遷移行列)で毎フレーム厳密伝播。積分誤差ゼロ。
  同じ関数で 1 周回先までのゴースト予測を生成。
- **チェイサーは自動ステーションキーピング**: 正直な HCW では Δv を与えた
  デブリは数分で km 単位漂流する。チェイサー(カメラ)が PD 制御で追従し、
  消費 Δv を燃料として表示。REALISM では燃料切れ = ミッション失敗
  (「相対ドリフトの放置はコスト」という実務と同じ教訓)。
- **R–V マップの原点は仮想 chief(デブリの元の軌道スロット)**: 推力を出し
  続けるチェイサーは CW フレームの原点として不適(推力体基準では相対運動が
  CW に従わない)。また追従制御下では「デブリ−チェイサー」は追従誤差(〜1 m)
  に潰れて何も見えない。教科書どおり原点 = 無推力の基準点(クロス表示)とし、
  チェイサーはマップに描かない(燃料メーターと 3D ビューが担当)。
- **勝利条件**: δa = 4x + 2ẏ/n(軌道エネルギー整合)、
  半径振動振幅 A = √((ẋ/n)² + (3x + 2ẏ/n)²)、近地点変化 δr_p = δa − A。
  δr_p ≤ −目標値(姿勢安定時)でクリア。タンブリングしたまま到達なら失敗
  (従来の UNCONTROLLED REENTRY を継承)。
  - ラジアルに押しても δa はその瞬間変わらない(式に ẋ が入っていない)
    = 「下に押しても軌道は下がらない」が HUD で即時に見える。
  - R–V 面の予測楕円の最下点 = δr_p なので「予測軌道がディスポーザル線に
    届けばクリア」という幾何的ゴールと数式が一致する。
- **ミッションの枠付け**: フルデオービット(600 km → 大気圏)は Δv ≈ 113 m/s で
  ゲーム時間では非現実的。現実のレーザー ADR 構想(LightForce 等)と同じく
  「1 パスで近地点を数 km 下げ、複数パス + 大気抵抗で除去」と位置づける。
  目標 Δ近地点: DEBRIS −2.0 km / BOX-WING −0.8 km / ROCKET −0.4 km
  (質量が大きいほど 1 パスの効果が小さい、も正直な教訓)。
- **レーザー推力**: 0.12 N(コサイン入射則)。実推定(mN 級)の約 100 倍に
  スケール(ゲームペーシングのため)。README に明記。質量は実規模
  (220 / 700 / 2600 kg)。
